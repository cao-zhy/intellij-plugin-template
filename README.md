# Template

IntelliJ IDEA 插件，添加一些文件模板和实时模板。

A plugin for IntelliJ IDEA, add some file templates and live templates.

## 更新日志 Changelog

### v1.1.0

初始化模块：

- 创建 README.md 文件
- 创建 .gitignore 文件
- 选择一个模板创建 license 文件

Initialize module with:

- Add a README.md file
- Add .gitignore
- Choose a license
